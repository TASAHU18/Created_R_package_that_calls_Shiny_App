How to test the package

1. To run the application, install the package from "library to be installed" folder which contains the .tar.gz file
   or   
   To build the package from scratch, execute the document, build and install commands to the package directory located in the shinyClusters folder
2. Load the library using the library("shinyClusters", lib.loc="~/R/win-library/3.4") command. lib.loc depends on your RStudio installation
3. Run the shiny app using the runClusterApplication() command
4. To view the function details (man page) enter the ?Function_name
